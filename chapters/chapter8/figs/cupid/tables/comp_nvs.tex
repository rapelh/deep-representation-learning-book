\caption{
\textbf{
Qualitative comparison on input view consistency.
}
We render the input view using its generated camera pose.
For view centric methods (LRM, LaRa), we use ground-truth intrinsic for rendering as they do not model intrinsic. 
%
%
Our method produces the highest-fidelity geometry and appearance; LRM hallucinates incorrect details, LaRa is overly blurry due to 2D diffusion inconsistencies, and 3D generation method OnePoseGen frequently fails to register pose reliably.
}
