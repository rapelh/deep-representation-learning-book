\textbf{Results for \textit{generative 3D reconstruction} from a single test image.} Given an input image (top left), \Cupid estimates camera pose (bottom left) and reconstructs 3D model (bottom right), re-rendering the input (top right). All results are produced in seconds via feed-forward sampling of the learned model.
